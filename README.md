## API/크롤링

### 교통카드빅데이터통합정보시스템
https://stcis.go.kr/

#### 노선별 이용량 ★
- 전국
- (노선, 일자) -> (정류장, 이용자유형, 시간대) -> 승차량

#### 노선별 차내재차인원 ★
- 전국
- (노선, 일자) -> (정류장, 이용자유형, 시간대) -> 재차인원

#### 정류장별 이용량
- 전국
- (정류장, 일자) -> (노선, 이용자유형, 시간대) -> (승차량, 하차량)

#### 정류장별 정차 노선수
- 전국
- (정류장, 일자) -> 노선

#### 노선별 혼잡도
- 전국
- (노선{광역}, 일자) -> (정류장, 시간대) -> 혼잡도

### 국가대중교통DB
https://kotsa.or.kr/ptc

#### 대중교통 이용지표

##### 노선별 이용인원
- 전국
- 연도 -> (노선, 요일) -> 이용객수

##### 정류장별 이용인원
- 전국
- 연도 -> (정류장, 요일) -> (승차량, 하차량)
